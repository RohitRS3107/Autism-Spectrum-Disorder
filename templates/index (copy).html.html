About

This application uses advanced face detection and classification algorithms to identify faces in uploaded images. You can upload an image, and the application will highlight faces and provide classification results.

How It Works

1. Choose an image file and upload it.
2. The system processes the image to detect faces.
3. Detected faces are highlighted and classified.
4. Results are displayed along with the processed image.
